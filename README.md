Weather API Client - Java

Description:
This Java application connects to the OpenWeatherMap API to fetch and display
current weather information for a given city.

Features:
- Fetches live weather data using HTTP connection
- Parses JSON response
- Displays temperature, feels-like temperature, and weather description

Technologies Used:
- Java
- OpenWeatherMap API
- JSON Library (org.json)

How to Run:
1. Replace YOUR_API_KEY with your OpenWeatherMap API key.
2. Compile the program:
   javac WeatherAPIClient.java
3. Run the program:
   java WeatherAPIClient

Note:
Internet connection is required to fetch weather data.
